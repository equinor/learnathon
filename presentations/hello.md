# 🎓 The Learnathon
## EDC 2026 — March 18
### Learn by building, reflect by sharing, win by exploring

-----

## What is the Learnathon?

A hands-on learning event where teams explore **AI-assisted software engineering** tools and methods by building something real — with security baked in throughout.

-----

## The Day at a Glance

| | Morning | Afternoon |
|---|---|---|
| 📚 | Learn the toolbox & workflow | 🔐 Security checkpoint |
| 🔨 | Mini-spec, plan & build | 🔨 Polish & prepare |
| ☕ | Coffee break | 🎤 Show & Tell |
| | | 🏆 Awards |

-----

## The Workflow

Mini-spec → Agent plan → Small commits → Continuous verification → Security checkpoint → Ship & share

-----

## Why?

- **Learn by doing** — not just watching presentations
- **Explore new tools** — Copilot, Claude Code, and more
- **Practice secure workflows** — security built in, not bolted on
- **Build community** — share experiences, learn from each other
- **Take it home** — repeatable patterns for your team

-----

| Tool | What It Does |

| **Spark** | Rapid prototyping via Copilot |

## 👥 Teams

- **3 people** per team (~20 teams)

## Pick Your Challenge

| Challenge | What You Build |
|-----------|---------------|
| 🎮 **Conference Stand Game** | Fun interactive game for EDC booth |
| 🔌 **Useful MCP Server** | Reusable tool for the community |

## 🎤 Show & Tell

The heart of the Learnathon — teams don't just demo, they **share what they learned**.

- 🧠 What surprised you?
- 🔧 What technique worked?
- ⚠️ What would you do differently?
- 💡 What will you take back to your team?

-----

## 🔐 Security — Built In, Not Bolted On

- Secrets hygiene — no API keys in code
- Dependency awareness
- Safe prompting and tool boundaries
- Recognizing AI hallucinations

-----
