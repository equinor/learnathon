- **imgflip.com/memegenerator** — classic meme templates with online captioning tool
- **kapwing.com/meme-maker** — drag-and-drop meme maker, supports custom fonts
- **knowyourmeme.com** — find the original template images, then caption in any editor

## Usage in slides

To embed in the markdown slides, place images in this `images/` folder and reference them:

```markdown
![Galaxy brain context engineering](/presentations/images/meme-galaxy-brain.png)
```

The reveal-bootstrap.js handles the base path automatically.
