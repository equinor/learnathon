# Learnathon Bingo App

An interactive bingo card web app for the Learnathon.

> **Status:** To be built — candidate for live demo using GitHub Spark or Claude Code

---

## What It Does

- Each team gets a digital bingo card (3×3 grid of techniques to try)
- Teams mark off squares as they complete them
- Optional: shared view on the big screen shows all teams' progress

---

## Build Plan

**Option A — GitHub Spark** (recommended for live demo)
- Build it in Spark during the 8:45 demo or at Trondheim test run
- Share a single URL for all teams
- Simple: Spark handles hosting

**Option B — Static HTML/JS** (Claude Code)
- Single HTML file, no server needed
- State per team stored in localStorage or URL params
- Works offline in Codespace
