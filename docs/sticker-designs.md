# Sticker Designs — EDC Secure Vibe Coding Learnathon

10 meme-infused sticker designs aligned with the Equinor Design System (EDS) palette. Die-cut, flat vector illustration style. Add text in post-production using the Equinor font (Bold 700 for titles, Medium 500 for subtitle).
